## Job-Cart Kanban Board


### 주요 기능 (Key Features) 
- 원클릭 공고 수집: 현재 보고 있는 채용 페이지의 제목과 URL을 자동으로 가져와 장바구니에 담습니다.
- 노션 스타일 칸반 보드: '담아둔 공고', '지원 완료', '종료' 3단계로 드래그 앤 드롭하여 상태를 관리합니다.
- 실시간 메모 편집: 보드 위에서 공고별 특이사항을 즉시 수정하고 저장할 수 있습니다.
- 커스터마이징: 보드 제목 변경 및 배경색 및 배경 이미지 설정 기능
- 스마트 정렬: 마감일순, 선호도순, 최신순으로 공고를 자유롭게 정렬합니다.
- 자동 마감 체크: 설정된 마감일이 지나면 자동으로 '종료' 컬럼으로 이동합니다.

###  기술 스택 (Tech Stack) 
Language: JavaScript (ES6+), HTML5, CSS3
API: Chrome Extension API (storage, tabs)
